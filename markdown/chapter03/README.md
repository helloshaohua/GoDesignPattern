# 什么是设计模式

- 是一套被反复使用、多数人知晓的、经过分类的、代码设计***经验的总结***。

## 类比武侠世界武功

- 设计模式 = 武功招式
- 基本原则 = 武功心法

## 设计模式的分类

- 创建型(单例模式、简单工厂模式、工厂方法模式、抽象工厂模式、建造者模式、原型模式)。
- 结构型(代理模式、适配器模式、装饰模式、桥接模式、组合模式、享元模式、外观模式)。
- 行为型(观察者模式、模板方法模式、命令模式、状态模式、职责链模式、解释器、中介者模式、访问者模式、策略模式、备忘录模式、迭代器模式)

## 本课程讲解的设计模式

- 创建型(简单工厂模式、工厂方法模式)。
- 结构型(装饰模式)。
- 行为型(策略模式)。
